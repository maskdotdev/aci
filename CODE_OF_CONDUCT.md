# Code of Conduct

ACI contributors are expected to keep discussions technical, respectful, and
focused on improving the project.

Unacceptable behavior includes harassment, threats, sustained personal attacks,
or publishing private information without permission.

Project maintainers may remove comments, close issues, or restrict participation
when conduct makes collaboration unsafe or unproductive. Report conduct issues
through the repository maintainers; for sensitive reports, use the same private
channel described in `SECURITY.md`.
